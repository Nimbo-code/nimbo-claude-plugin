Claude Code Plugin

Fine-tune LLMs.

A Claude Code plugin that turns natural language into an end-to-end ML pipeline. From LoRA fine-tuning to CoreML conversion to on-device deployment.

Get Started Framework
Claude Code
# Add the Nimbo marketplace
$ claude plugin marketplace add https://github.com/Nimbo-code/nimbo-claude-plugin
✔ Successfully added marketplace: nimbo
 
$ claude plugin install nimbo
✔ Successfully installed plugin: nimbo@nimbo

Features

Everything you need to ship

🎯

LoRA & QLoRA

Fine-tune any supported model with LoRA or 4-bit QLoRA. Automatic hardware detection picks the best configuration for your GPU.

🚀

Triton Kernels

Custom Triton kernels for RMSNorm (7-8x), SwiGLU (3-5x), and RoPE (1.9-2.3x) speedups. One flag to enable.

🍎

CoreML Export

Convert to CoreML with LUT quantization (4/6/8-bit). Optimized for Apple Neural Engine on iPhone and Mac.

💬

Natural Language

No API docs to memorize. Describe what you want in natural language and Claude selects the right skill automatically.

📊

15+ Architectures

LLaMA, Mistral, Phi, Qwen2, EXAONE, GPT-2, Falcon, BLOOM, Gemma, and more. Full LoRA support across all.

🛠

Config Management

Generate, save, and load YAML configs. Share training setups across your team with a single file.

Performance

Usage

Just say what you need

Claude automatically matches your intent to the right skill. No commands to learn.

"Install Nimbo and set up my environment"
setup → Python, venv, GPU detection
"Prepare my JSONL data for instruction tuning"
dataset → Format, template, validate
"Fine-tune LLaMA 3.2 1B with QLoRA"
train → 4-bit LoRA, auto-config
"Test the model with streaming output"
inference → Generate, stream, chat
"Convert to CoreML for my iPhone"
export → LUT quantization, .mlpackage
"Speed up training with Triton kernels"
kernel → RMSNorm, SwiGLU, RoPE
